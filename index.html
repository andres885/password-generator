Recomendaciones para MariaDB:

  • Usa siempre comillas simples al asignar la contraseña: IDENTIFIED BY 'tu_contraseña'
  • Longitud mínima recomendada: 12 caracteres
  • Caracteres problemáticos como & y ; deben escaparse o evitarse
  • Esta herramienta genera contraseñas compatibles por defecto

Comando SQL Listo

CREATE USER 'nuevo_usuario'@'localhost' IDENTIFIED BY 'tu_contraseña';

Ejemplos de uso:

-- Crear nuevo usuario
CREATE USER 'admin'@'localhost' IDENTIFIED BY 'tu_contraseña';

-- Cambiar contraseña de usuario existente
SET PASSWORD FOR 'admin'@'localhost' = PASSWORD('tu_contraseña');

-- Con mysqladmin desde terminal
mysqladmin -u root -p password 'tu_contraseña'
